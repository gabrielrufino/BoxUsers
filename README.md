# UBox

A simple and intuitive user system

## Install

<strong>Exec:</strong>
<pre>
git clone https://github.com/gabrielrufino/UBox/
cd UBox
npm install
</pre>

## Start MongoDB

### Windows

<strong>Exec:</strong>
<pre>
mongod
</pre>

## UBox Ready!

<strong>Exec:</strong>
<pre>
npm start
</pre>

Access http://localhost:3000! UBox is ready.
